# Лабораторная работа №2 — Структуры в языке C

## Описание

Изучение объявления и использования структур, вложенных структур, указателей на структуры, массивов структур, передачи структур в функции, перечислений (`enum`) и объединений (`union`).

## Содержание

| Файл  | Задание |
|-------|---------|
| `1.c` | Базовая структура `Student`: объявление, заполнение, ввод с клавиатуры |
| `2.c` | Массив из 5 структур `Student`, поиск студента с максимальным баллом |
| `3.c` | Вложенные структуры: `Date` внутри `Person`, фильтрация по году рождения |
| `4.c` | Структура `Rectangle`, функции площади и периметра (по значению и по указателю) |
| `5.c` | Перечисление `Day` (дни недели), `switch`, `enum` внутри структуры |
| `6.c` | Объединение `union Data`, демонстрация перезаписи, tagged union |
| `7.c` | Итоговое: массив студентов с `enum Course`, добавление, вывод, поиск, сортировка |

## Технологии

- Язык: **C** (стандарт C11)
- Компилятор: **gcc**

## Сборка и запуск

Скомпилировать отдельный файл:

```bash
gcc -std=c11 -Wall 1.c -o 1
./1
```

Скомпилировать все сразу:

```bash
for f in *.c; do gcc -std=c11 -Wall "$f" -o "${f%.c}"; done
```
