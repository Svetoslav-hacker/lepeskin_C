# Лабораторная работа №1 — Типы данных, память и выражения в C

## Описание

Изучение базовых типов данных языка C, работы с памятью, арифметических и логических операций, ввода/вывода, переполнения и потери точности.

## Задания

| Файл        | Описание                                      |
|-------------|-----------------------------------------------|
| task01.c    | Переменные int, float, char и их вывод        |
| task02.c    | Размеры типов (sizeof)                        |
| task03.c    | Значения переменных и их адреса в памяти      |
| task04.c    | Остаток от деления (%)                        |
| task05.c    | Инкремент и декремент (++, --)                |
| task06.c    | Целочисленное деление и преобразование типов  |
| task07.c    | Потеря точности float (0.1 + 0.2)             |
| task08.c    | Переполнение int (INT_MAX + 1)                |
| task09.c    | Вычисление формул x²+2x+1 и (x+1)²-(x-1)²   |
| task10.c    | Обмен значений двух переменных                |
| task11.c    | Последняя цифра числа                         |
| task12.c    | Разложение трёхзначного числа (сотни/десятки/единицы) |
| task13.c    | Перевод секунд в формат HH:MM:SS              |
| task14.c    | Исследование адресов переменных в памяти      |

## Технологии

- Язык: **C** (стандарт C99)
- Компилятор: **gcc**

## Сборка и запуск

Скомпилировать отдельный файл:

```bash
gcc -std=c99 -Wall taskXX.c -o taskXX
./taskXX
```

Скомпилировать все сразу:

```bash
for f in task*.c; do gcc -std=c99 -Wall "$f" -o "${f%.c}"; done
```
